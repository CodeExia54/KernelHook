Notes:
- KernelSU did not ship an android14-5.10 build for this release; Pixel_31
  AVD (which Google ships with kernel 5.10) is exercised against the
  android12-5.10 LKM. The KH path-1 / path-2 paths only require ABI/vermagic
  match with the running kernel — KSU's "android version" suffix is for
  KSU's own userspace bring-up and irrelevant to KH's loader.
- android13-5.15 / android14-5.15 / android14-6.1 are released by KSU but
  not currently exercised by KH AVDs; add a directory + entry above when a
  matching AVD lane lands.
- The .ko binaries are NOT checked into the repo. Run khtools/fixtures/ksu_lkm/fetch.sh
  to materialize them locally; CI also runs fetch.sh before path-1/path-2 lanes.
